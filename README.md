# How to Use

### Create environment
Bikin env terlebih dahulu dengan nama env
`python -m venv env`

### Install library
Install library yang dibutuhkan
```
python pip install -r requirements.txt
```

### Run program
Jalankan program 
```
python manage.py runserver
```

### API
```
http://127.0.0.1:8000/api/
```
method `post`
parameter `query`
contoh: `query:1*5+(7-1)`
